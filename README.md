# Aira ERP

## Overview
Aira ERP is an integrated system designed to manage various aspects of an organization effectively, including finance, operations, projects, and more.

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/funeducationtv/aira-erp.git
   cd aira-erp
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Create a `.env` file for environment variables and update it based on the `.env.example` file.

## Usage Guidelines
- To start the application, use the following command:
  ```bash
  npm start
  ```
- Access the application at `http://localhost:3000`.
